# CrowdSense AI 🚀

An AI-powered crowd detection and monitoring web application built using Flask, OpenCV, and Computer Vision techniques. The project helps analyze crowd density in real-time using video or camera input.

---

## 📌 Features

- 👥 Real-time crowd detection
- 📷 Live camera integration
- 🎥 Video/image crowd analysis
- 📊 Crowd monitoring dashboard
- 🧠 AI & Computer Vision based detection
- 🌐 Interactive web interface
- ⚡ Fast processing with OpenCV

## ⚙️ Installation & Setup

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/YOUR_USERNAME/YOUR_REPOSITORY_NAME.git
```

### 2️⃣ Move into Project Folder

```bash
cd YOUR_REPOSITORY_NAME
```

### 3️⃣ Install Dependencies

```bash
pip install -r requirements.txt
```

### 4️⃣ Run the Flask Application

```bash
python app.py
```

---

## 🚀 Usage

1. Start the Flask server
2. Open browser at:

```bash
http://127.0.0.1:5000/
```

3. Upload image/video or start live detection
4. Analyze crowd activity in real time
